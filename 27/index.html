Удаление элемента из массива

Описание задачи:

Дан целочисленный массив nums и целое число val. Требуется удалить все вхождения val в массиве nums "на месте". Порядок элементов может быть изменен. Функция должна вернуть количество элементов в nums, которые не равны val.

Требования:

  • Изменить массив nums так, чтобы первые k элементов содержали элементы, не равные val
  • Оставшиеся элементы массива nums не важны
  • Вернуть значение k (количество элементов, не равных val)

Пример:

Вход: nums = [3,2,2,3], val = 3
Выход: 2, nums = [2,2,_,_]
Пояснение: Функция должна вернуть k = 2, где первые два элемента nums равны 2.
Неважно, что останется после возвращаемого k (поэтому они отмечены символом подчеркивания).
